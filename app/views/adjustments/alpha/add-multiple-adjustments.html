Adding multiple adjustments

Process

  1. User selects required adjustment from the list
  2. User enters data related the adjustment
  3. User reviews data entered on previous scree, user has opportunity to:
    • Add another adjustment
    • Edit any data entered in error
    • Confirm and continue
  4. After selecting add another adjustment user is taken to an adjustment type selection page
  5. User then enters data for selected adjustment
  6. User reviews data entered on previous scree, user has opportunity to:
    • Add another adjustment
    • Edit any data entered in error
    • Confirm and continue
  7. After selecting continue the user is returned to the dashboard that has been updated
    • Notification panel confirms data has been saved and plays back a summary of the data added
    • Card relevant to adjustment has been updated with a new count
    • CTA's for calculate release dates are present
