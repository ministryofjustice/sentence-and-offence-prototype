Before you can continue to this service, you need to check if any offences without a sentence have been replaced.

This service has flagged that there is at least one offence that does not have a sentence imposed. This can happen if an offence was replaced (for example, reduced to a lesser offence).

To enable this service to accurately calculate relevant remand, check if the offence was replaced.

Select no if there is any other reason that the offence does not have a sentence imposed.

Why is this important?

Time spent on remand is marked as not relevant if the offence did not result in a sentence being imposed. 

However, if the offence was replaced, and a sentence was imposed for that replaced offence, this remand might be relevant.
